## 📍 placepic service

 <b>우리들끼리 공유하는 최애 장소, 플레이스픽 </b>

 플레이스픽은 신뢰있는 커뮤니티 기반의 장소 정보 공유 플랫폼입니다. 

 **Service key Feature**
  * Exploration - 쉽고, 빠르게 원하는 장소를 탐색

  * Discovery - 가보고 싶은 장소를 발견할 수 있어야함

  * Interaction - 상호작용, 소통할 수 있는 공간
